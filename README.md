For start your Platform9:
1. Install latest npm and nodeJS. 
'sudo apt-get install nodejs nodejs-legacy npm'
2. Install All Libs...
'npm install'
3. Run IT:
'env PORT=[YOUR PORT] npm start'
Exmaple:
'env PORT=80 npm start'
